12. ziarah kubur juga mengingatkan tentang adanya balasan di..........​


Ziarah kubur juga mengingatkan tentang adanya balasan di akhirat atau kehidupan setelah kematian. Dalam Islam, ziarah kubur merupakan salah satu cara untuk mengingatkan diri tentang kematian dan akhirat serta untuk memohon ampunan dan rahmat Allah SWT bagi orang yang telah meninggal dunia. Selain itu, ziarah kubur juga dapat memperkuat rasa ikhlas dan kesadaran akan kefanaan dunia serta menumbuhkan motivasi untuk memperbanyak amal sholeh sebelum ajal menjemput. Dalam pandangan agama-agama lainnya, ziarah kubur juga memiliki makna yang serupa, yaitu untuk mengingatkan manusia tentang kematian dan balasan di kehidupan setelahnya.

Perpangkatan adalah operasi matematika untuk perkalian berulang dengan bilangan yang sama sebanyak pangkatnya.

✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤✤

[tex]{\boxed{\bold{ \pink{Bentuk \: Umum\: }}}}[/tex]

[tex]\begin{gathered}\text{a}^ \text{n} = \underbrace{\text{a × a × a×...×a}}_{\text{n faktor}} \\ \end{gathered}[/tex]

[tex]{{\bold{ { a \times a \times a... = {a}^{n} \: }}}}[/tex]

Keterangan :

a = basis

n = pangkat

[tex]{\boxed{\bold{ \pink{Syarat \: Bilangan \: Berpangkat\: }}}}[/tex]

[tex]{{\bold{ { {a}^{m} \div {a}^{n} = {a}^{m - n} \: }}}}[/tex]

[tex]{{\bold{ { {a}^{m} \times {a}^{n} = {a}^{m + n} \: }}}}[/tex]

[tex]{{\bold{ {( {a}^{m}) ^{n} = {a}^{m \times n} \: }}}}[/tex]

[tex]{{\bold{ { \sqrt[n]{ {a}^{m} } = a \frac{m}{n} \: }}}}[/tex]

[tex]{{\bold{ { (ab) ^{n} = {a}^{n} {b}^{n} \: }}}}[/tex]

[tex]{{\bold{ {a ^{ - n} = \frac{1}{ {a}^{n} } \: }}}}[/tex]

[tex]{{\bold{ { {a}^{0} = 1 \: }}}}[/tex]

[tex]{{\bold{ { {a}^{1} = a, a ≠ 0 \: }}}}[/tex]

• a, b, m, n ∈ R

[tex] \large \sf {2}^{2} \div {2}^{2} [/tex]

[tex] \large \tt = {2}^{2 - 2} [/tex]

[tex] \large \sf = {2}^{0} [/tex]

[tex] \large \sf = 1[/tex]

KESIMPULAN

Hasil dari 2² ÷ 2² adalah 1
