2. bagaimana cara mendengarkan lagu di aplikasi joox sedang offline


1.) Download aplikasinya
2.) terus log in, disaranin log in lewat facebook atau gak we chat
3.) abis itu pilih lagunya (gak usah diklik)
4.) nah kalo yg dihape itu, disebelah kanan ada icon (titik2gitu) terus diklik
5.) abis itu nanti ada pilihan unduh/tambah/dll
6.) pilih yg unduh. biasanya joox itu nyarani unduh/downloadnya pake jaringan wifi
7.) kalo cara lain itu yg lagunya diklik dulu
8.) terus lihat paling bawah ada icon2 kayak ( download, bintang dll)

22. Bagaiman cara menentukan chord sebuah lagu?


Cara untuk menemukan kunci gitar pada umunya dalam sebuah lagu ada dua cara, yaitu :
1. Dengan Nada dari sebuah Lagu
Mencari kunci gitar atau chord sebuah lagu dapat dicari dari nada dari lagu tersebut. Biasanya lagu yang dicantumkan nadanya adalah lagu – lagu klasik, nasional, rohani, dan lain – lain.
2. Memakai Feeling atau perasaan

Metode ini sangat sering dikatakan oleh pemain gitar yang sudah sering dalam bermain gitar untuk mencari kunci gitar atau chord dalam sebuah lagu. Sebenarnya memakai feeling atau perasaan ada rumusan tertentu yang membantu pemain gitar dalam mencari kunci gitar sehingga pemain gitar dapat mengiringi lagu yang diinginkan.

25. Yang merupakan Software Aplikasi adalah.....


Pengertian software aplikasi, yaitu suatu pedrangkat lunak berupa program komputer yang berfungsi untuk melakukan tugas-tugas khusus, seperti membuat dokumen contoh aplikasinya mickroskop word, mengedit foto  contohnya potoshop , membuat laporan keuangan, atau aplikasi hibyran seperti game.

Pengertian software aplikasi tidak lepas dari beberapa macam program pembangunnya, yaitu terdiri dari software hiburan, pendidikan, bisnis, perangkat lunak khusus, serta produktivitas kerja. 

Perangkat lunak penunjang produktivitas kerja memberikan peranan yang sangat bermanfaat untuk optimalisasi mutu kerja.

Sebuah data yanga di program dan disimpan secara digital yg tdk terlihat secara fisik tetapi dlm komputer

37. Dibawah ini adalah software pembuat presentasi offline, kecuali


1. Microsoft PowerPoint merupakan sebuah program komputer untuk presentasi yang dikembangkan oleh Microsoft di dalam paket aplikasi kantoran mereka, Microsoft Office (Office 365), selain Microsoft Word, Excel, Access dan beberapa program lainnya.

2. Google Slides adalah program presentasi yang disertakan sebagai bagian dari paket office perangkat lunak berbasis web gratis yang ditawarkan oleh Google dalam layanan Google Drive-nya. Layanan ini juga mencakup Google Docs dan Google Sheets, pengolah kata dan spreadsheet masing-masing.

3. Prezi Viewer merupakan aplikasi android untuk mengedit dan menampilkan dokumen Power Point via smartphone Android. Dengan menghubungkan Aplikasi Prezi Viewer dengan Chromecast maka kita dapat menampilkan slide presentasi dengan mudah dan cepat. Tak hanya itu saja, selain menampilkan, kita juga dapat mengedit atau melakukan kostumisasi sesuai keinginan.

4. OfficeSuite + PDF Editor. Sama seperti aplikasi sebelumnya, aplikasi ini memiliki keunggulan dengan fitur yang hampir sama dengan aplikasi Polaris Office + PDF. Aplikasi ini memiliki fitur ekslusif dengan tampilan unik dan menarik.

5. Presentation Link adalah aplikasi presentasi sederhana yang tentunya dapat membantu anda dalam menampilakan slide dengan mudah dan efisien. Aplikasi ini cocok bagi seorang mahasiswa yang mendapat tugas presentasi serta bagi kalangan pebisnis.

6. Join.Me merupakan aplikasi android untuk keperluan presentasi jarak jauh. Dengan aplikasi ini, anda tetap bisa melakukan presentasi dimana saja dan kapan saja, entah itu di bandara, di rumah, kedai kopi, ataupun tempat lainnya.

7. Visme menawarkan semua alat yang Anda butuhkan untuk presentasi. Tak hanya sekedar aplikasi untuk presentasi biasa, Visme juga memberikan infografis, visualisasi data, laporan, demo produk, dan resume. Kita pun bisa menggunakan template yang memang disediakan oleh aplikasi ini. Tak hanya itu saja, tampilan background HD dan layout yang disajikan oleh Visme tampak sangat profesional. Aplikasi ini dikemas dalam jutaan gambar gratis, ribuan ikon vektor, alat grafik, dan ratusan jenis font. Kita bisa membuat konten visual dengan Visme dengan menambahkan animasi, interaktivitas penuh, dan audio.

8. Emaze Merupakan sebuah alat presentasi sederhana yang menggunakan HTML5 ini bisa berjalan pada setiap browser maupun perangkat yang berbeda, termasuk Chromebook atau merk tablet lainnya. Emaze dirancang khusus untuk para pengguna yang ingin membuat presentasi mudah hanya dalam waktu beberapa menit saja.
