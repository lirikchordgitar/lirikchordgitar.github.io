31. Pernyataan yang ekuivalen dengan pernyataan “jika semua siswa hadir, maka beberapa guru tidak hadir” adalah ... a. beberapa siswa tidak hadir atau beberapa guru hadir b. semua siswa tidak hadir atau beberapa guru tidak hadir c. beberapa siswa tidak hadir atau beberapa guru tidak hadir d. beberapa siswa tidak hadir atau beberapa guru tidak hadir e. semua siswa hadir dan beberapa guru hadir (Ujian Nasional UN 2014)


p = semua siswa hadir
q = beberapa guru tidak hadir

p → q ≡ ~q  → ~p ≡ ~p ∨ q

Untuk kesetaraan yang pertama :
~q = semua guru hadir
~p = beberapa siswa tidak hadir

kalimatnya menjadi :
"Jika semua guru hadir, maka beberapa siswa tidak hadir"


Untuk kesetaraan yang kedua :
~p = beberapa siswa tidak hadir
q = beberapa guru tidak hadir

Kalimatnya menjadi :
"Beberapa siswa tidak hadir atau beberapa guru tidak hadir"

Sehingga, jawabannya adalah C
